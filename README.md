# Eva-Liter-ria
IA para Escritores é um assistente literário que ajuda autores a criar histórias, personagens e capítulos, revisar textos, traduzir conteúdos e organizar suas obras. Com efeito de digitação realista e botões de pause/continuar, a IA inspira criatividade e otimiza sua escrita de forma prática e interativa.
